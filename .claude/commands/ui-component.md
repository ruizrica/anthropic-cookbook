# UI Component Command

Generate production-ready UI components with full documentation and tests.

**Component Generation**:

1. **Component Code**:
   - React/Vue/Angular/Svelte
   - TypeScript interfaces
   - Props validation
   - Accessibility (ARIA)
   - Keyboard navigation

2. **Styles**:
   - CSS Modules / Styled Components / Tailwind
   - Responsive design
   - Dark mode support
   - Animation/transitions

3. **Variants**:
   - Size variants (sm, md, lg)
   - Visual variants (primary, secondary, outline, ghost)
   - State variants (disabled, loading, error)

4. **Tests**:
   - Unit tests (Jest/Vitest)
   - Accessibility tests (jest-axe)
   - Visual regression tests (Chromatic)

5. **Documentation**:
   - Storybook stories
   - Props documentation
   - Usage examples
   - Accessibility notes

Use the `component-library-builder`, `modern-frontend-design`, and `web-accessibility-auditor` skills.

**Supported Components**:
- Button, Input, Select, Checkbox, Radio
- Modal, Drawer, Tooltip, Popover
- Card, Table, List
- Tabs, Accordion, Dropdown
- Form components
- Navigation components

**Best Practices Applied**:
- Semantic HTML
- WCAG 2.1 AA compliance
- Keyboard accessibility
- Screen reader support
- Touch-friendly (44x44px targets)
- Responsive design
- Design system tokens
